# SPX_LLM_VISION_TRADER

Full starter project generated by ChatGPT.

Important architecture:

- TradingView chart = visual battlefield
- Playwright = camera / eyes
- Google Sheet = live CALL and PUT evidence
- Python = controller / watcher / loop runner only
- LLM Vision = only brain

Python only moves evidence, watches LLM-created triggers, loops during battle, stores history, and reports LLM output. It does not create trading intelligence.

Files are also provided as a downloadable zip in the ChatGPT response because some GitHub file writes can be blocked by connector safety checks when a file contains secret-style environment variable names.

## Quick Windows setup

```powershell
cd SPX_LLM_VISION_TRADER
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
copy .env.example .env
python main.py --test-strict
python main.py --test-db
python main.py --test-screenshot
python main.py --test-sheets
python main.py
```

Fill `.env` with your LLM provider, LLM model, TradingView URL, Google Sheet ID, CALL tab, PUT tab, and Google service account JSON path.
